# Starfield Animation

A mesmerizing particle animation with a twinkling starfield background created with Processing.


## How to Run

1. Install [Processing](https://processing.org/)
2. Download or clone this repository
3. Open `star.pde` in Processing
4. Click the Play button


## Customization

You can tweak the animation by modifying these variables in `star.pde`:

- `particles_num` - Number of moving particles (default: 20)
- `bgStars_num` - Number of background stars (default: 200)
- `speed` - Particle movement speed (default: 5.0)
- `particle_size` - Size of particles (default: 4.0)
- `flash_interval` - Frames between star twinkles (default: 60)
